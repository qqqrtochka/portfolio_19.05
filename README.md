# Одностраничник (портфолио)

## Как открыть

- Просто откройте `index.html` в браузере.
- Или запустите локальный сервер:
  - `python -m http.server 5173`
  - затем откройте `http://localhost:5173`

## Что менять

- Текст/имя/контакты — в `index.html`
- Стили — в `main.css`
- Небольшой JS (плавная прокрутка, копирование email) — в `js/main.js`
